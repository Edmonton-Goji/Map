Welcome to the Edmonton Goji Map

Select a goji on the map for more information or use the menu to:

1) Search for a goji by its name, address, or neighbourhood.

2) View the photo gallery. Select an image to view it in fullscreen (desktop only) or select the plant name for more information.

3) Add a goji to the map.
